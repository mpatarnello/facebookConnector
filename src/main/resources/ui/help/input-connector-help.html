This is a Facebook demo app.

Note: Integrations using the Facebook components can only be run on FoxWeave infrastructure i.e. you cannot use them On-Premise or On-PaaS. This is because of the type of authentication used by the Facebook service (OAuth v2), which requires the use of a "consumerSecret", which must be protected and so cannot be exposed to non-FoxWeave infrastructure (i.e. On-Premise or On-PaaS machines).
